# InsightPilot 用户使用手册

## 1. 注册与登录

首次使用进入注册页，填写用户名和不少于 6 位的密码。注册成功后返回登录页。

## 2. 上传数据

进入“数据集”页面，选择“上传数据”。系统支持 CSV、XLS 和 XLSX 文件。上传成功后自动计算行列数、字段类型、缺失值、重复值、异常值和健康度。

## 3. 数据清洗

在数据集详情页进入“数据清洗”，选择操作和字段：

- 删除重复数据
- 删除含缺失值的行
- 均值、中位数、众数或固定值填补
- 文本前后空格清理
- IQR 异常值删除
- Z-score 标准化
- Min-Max 标准化

每次操作都会保存清洗记录，并生成新的 CSV 版本。

## 4. 手动统计分析

“统计分析工作台”支持描述统计、频数统计、分组聚合、相关分析、线性回归和 K-Means 聚类。选择分析类型与字段后点击“执行分析”，系统保存结果和图表配置。

## 5. AI 数据助手

输入自然语言问题，例如：

- 按专业统计平均成绩并绘制柱状图
- 分析学习时长和成绩之间的关系
- 对学习时长和出勤率进行三类聚类

系统先生成 JSON 分析计划，再通过白名单校验，最后由 Pandas 或 Scikit-learn 计算。模型不直接生成最终统计数字，也不执行任意代码。

## 6. 报告生成

在数据集详情页点击“生成报告”。报告包含数据概况、质量指标、已保存的分析任务和 AI 对话摘要，可导出为 HTML。

## 7. 数据安全提示

不要上传含有身份证号、银行卡号、密码等敏感信息的数据。使用云端大模型接口时，应先确认数据字段是否允许发送到第三方服务。
