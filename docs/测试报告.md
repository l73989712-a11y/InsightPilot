# InsightPilot 系统测试报告

## 1. 测试范围

覆盖用户认证、数据上传、CSV/Excel 读取、质量检测、数据清洗、统计分析、AI 计划生成、安全校验、报告生成和权限隔离。

## 2. 自动化测试

项目包含：

- `tests/test_analysis_service.py`：分析算法单元测试
- `tests/test_integration.py`：注册、登录、上传、分析、AI 和报告的完整流程测试

执行命令：

```bash
python -m pytest -q
```

当前工程在构建环境中运行结果为 4 项测试全部通过。
