# InsightPilot 部署与安装手册

## 1. 推荐环境

- Windows 10/11 或主流 Linux
- Python 3.10 或 3.11
- MySQL 5.7
- Chrome 或 Edge
- 可选：兼容 OpenAI Chat Completions 的大模型接口

## 2. 安装流程

1. 解压项目，进入 `InsightPilot` 目录。
2. 创建并激活虚拟环境。
3. 执行 `pip install -r requirements.txt`。
4. 在 MySQL 中执行 `sql/init.sql`。
5. 将 `.env.example` 复制为 `.env`，填写数据库密码。
6. 执行 `flask --app run.py init-db`。
7. 执行 `python run.py`。
8. 浏览器访问 `http://127.0.0.1:5000`。

## 3. Windows 一键辅助脚本

双击或在命令行运行 `scripts/start_windows.bat`。首次运行会建立虚拟环境并生成 `.env`。修改数据库密码后再次运行即可。

## 4. Linux 辅助脚本

```bash
chmod +x scripts/start_linux.sh
./scripts/start_linux.sh
```

## 5. 快速演示模式

没有 MySQL 时，可先运行：

```bash
python scripts/run_demo_sqlite.py
```

该模式只用于界面预览和功能验证，正式提交仍应使用 MySQL 5.7。

## 6. 大模型配置

离线规则模式：

```env
LLM_PROVIDER=rule
```

兼容接口模式：

```env
LLM_PROVIDER=openai_compatible
LLM_API_KEY=你的密钥
LLM_BASE_URL=https://服务地址/v1
LLM_MODEL=模型名称
```

## 7. 常见问题

- 数据库连接失败：检查 MySQL 服务、账号、密码、端口和数据库名。
- 中文 CSV 乱码：系统依次尝试 UTF-8、GBK、GB18030；仍失败时请另存为 UTF-8 CSV。
- 图表不显示：确认 `app/static/vendor/plotly.min.js` 完整存在。
- Excel 读取失败：确认已安装 `openpyxl` 和 `xlrd`。
- 上传失败：检查文件大小是否超过 `.env` 中的限制。
