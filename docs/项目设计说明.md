# InsightPilot 项目设计说明

## 1. 项目名称

基于 Flask 与大语言模型的智能数据分析与可视化平台设计与实现。

## 2. 项目目标

面向缺乏编程经验的普通用户，提供数据上传、质量检测、清洗、统计分析、图表展示、自然语言分析和自动报告生成服务。

## 3. 系统角色

- 普通用户：管理自己的数据集、执行分析、使用 AI 助手、生成报告。
- 管理员：作为扩展功能，可查看系统运行与用户情况。

## 4. 系统模块

1. 用户认证模块
2. 数据集管理模块
3. 数据质量检测模块
4. 数据清洗模块
5. 统计分析模块
6. 数据可视化模块
7. AI 数据助手模块
8. 报告生成模块

## 5. 安全设计

系统不执行大模型生成的 Python 代码，而是要求模型生成结构化 JSON 分析计划。后端只接受白名单动作，并检查字段名、聚合方法、聚类数量等参数。

## 6. 推荐答辩演示流程

- 上传学生学习数据。
- 展示健康度、缺失值、重复值和异常值。
- 完成一次数据清洗。
- 生成描述统计与分组聚合图。
- 使用自然语言提问。
- 展示模型计划、真实计算结果和 AI 总结。
- 生成 HTML 报告。
